# Device Lifecycle Management
**SnowTelco Communications - Operations**

## Purpose

This document defines the policies and procedures for managing the complete lifecycle of mobile devices from procurement through disposal.

---

## Scope

This policy covers:
- Smartphones and feature phones
- Tablets
- Mobile broadband devices (MiFi)
- Wearables (smartwatches)
- IoT devices

---

## Device Lifecycle Stages

```
Procurement → Inventory → Fulfilment → Active Use → Return/Upgrade → Refurbishment/Disposal
```

## 4. Active Use Support

### Warranty Coverage

| Coverage Type | Duration | Provider |
|---------------|----------|----------|
| Manufacturer warranty | 12-24 months | OEM |
| Extended warranty | Up to 36 months | SnowTelco |
| Device protection | Ongoing subscription | Insurance partner |

### Warranty Claims

**Process:**
1. Customer reports fault
2. Technical triage
3. Claim assessment
4. Repair/replacement decision
5. Fulfilment
6. Vendor back-claim (if applicable)

### Repair vs Replace

| Scenario | Action |
|----------|--------|
| Manufacturing defect | Replace (warranty) |
| Accidental damage (insured) | Repair or replace |
| Accidental damage (uninsured) | Customer pays |
| Software issue | Technical support |
| Out of warranty | Customer options |

### Repair Partners

| Partner | Services | SLA |
|---------|----------|-----|
| Apple Authorised | iPhone repairs | 3-5 days |
| Samsung Authorised | Samsung repairs | 3-5 days |
| Third-party (Assurant) | All brands | 2-3 days |

## 5. Device Returns

### Return Reasons

| Reason | Return Window | Condition Required |
|--------|---------------|-------------------|
| Change of mind | 14 days | As new |
| Faulty (DOA) | 30 days | Any |
| Upgrade return | 14 days after new device | Working |
| End of contract | Any time | Any |

### Return Process

1. Customer initiates return
2. Returns label generated
3. Device received at returns centre
4. Inspection and grading
5. Credit/refund processing
6. Device routing (refurb/recycle)

### Grading Criteria

| Grade | Condition | Value Retention |
|-------|-----------|-----------------|
| A | Like new, no marks | 80-90% |
| B | Minor cosmetic marks | 60-75% |
| C | Visible wear, fully functional | 40-55% |
| D | Functional with defects | 20-35% |
| F | Non-functional | Recycle only |

---

## 6. Refurbishment

### Refurbishment Process

1. Receiving and logging
2. Data wipe (ADISA certified)
3. Functional testing
4. Cosmetic assessment
5. Repair (if needed)
6. Quality assurance
7. Repackaging
8. Return to stock or sale

### Data Security

**Data Wipe Standards:**
- ADISA certified process
- Certificate of destruction
- 100% audit trail
- GDPR compliant

### Refurbished Device Sales

| SnowTelco Refurbished | Budget customers |
| Trade partner | Wholesale market |
