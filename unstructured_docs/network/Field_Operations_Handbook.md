# SnowTelco Field Operations Handbook

## Introduction

This handbook provides comprehensive guidelines for field technicians conducting installations, repairs, and maintenance activities across SnowTelco's network.

## Safety First

### Personal Protective Equipment (PPE)
- High-visibility vest (always)
- Safety boots with steel toe caps
- Hard hat (when working at height or on construction sites)
- Safety glasses (when working with cables/equipment)
- Gloves (electrical work, cold weather)

### Working at Height
- Harness required for work above 2 meters
- Ladder inspections before each use
- Two-person rule for mast climbing
- Weather restrictions: No work in winds >40mph

### Electrical Safety
- Lock-out/tag-out procedures mandatory
- Test before touch
- Assume all circuits are live until verified
- Never work alone on high-voltage equipment

## Work Order Management

### Work Order Types

| Type | SLA | Priority |
|------|-----|----------|
| Emergency Repair | 4 hours | P1 |
| Service Restoration | 8 hours | P2 |
| Installation | Next business day | P3 |
| Planned Maintenance | Scheduled window | P4 |

### Work Order Lifecycle

1. **Dispatch** - Received via mobile app
2. **En Route** - Mark when traveling
3. **On Site** - Check in at location
4. **In Progress** - Work commenced
5. **Complete** - Job finished, customer sign-off
6. **Closed** - Documentation submitted

### Mobile App Usage

- Check work orders at start of shift
- Update status in real-time
- Capture photos of completed work
- Record parts used
- Obtain customer signature

## Installation Procedures

### Consumer Mobile (SIM/eSIM)
1. Verify customer identity (passport/driving license)
2. Activate SIM in provisioning system
3. Configure device settings
4. Test voice, SMS, data
5. Explain billing and support options
6. Obtain satisfaction confirmation

### Business Installation
1. Site survey completed prior to visit
2. Equipment inventory check
3. Install hardware per design document
4. Cable management to standard
5. Configure and test all services
6. Handover to customer with documentation

### Network Element Installation
1. Follow network design specifications
2. Power requirements verified
3. Backhaul connectivity tested
4. Element registered in inventory system
5. Integration testing with NOC
6. Acceptance sign-off

## Repair Procedures

### Diagnostic Steps
1. Review ticket history
2. Remote diagnostics (if possible)
3. On-site visual inspection
4. Systematic fault isolation
5. Root cause identification
6. Repair or escalate

### Common Fault Resolutions

| Fault | Likely Cause | Resolution |
|-------|--------------|------------|
| No signal | Antenna issue, coverage gap | Check antenna, report coverage |
| Slow data | Congestion, backhaul | Check cell load, escalate |
| Call drops | Handover failure | Check neighbor relations |
| No service | SIM issue, account | Verify account status |

### Escalation Criteria
- Unable to resolve within SLA
- Safety concern
- Requires specialist equipment
- Customer complaint (P1/P2)

## Parts and Inventory

### Van Stock
- Maintain minimum stock levels
- Daily van check before shift
- Report low stock via app
- Return unused parts to warehouse

### Parts Ordering
- Order via mobile app
- Next-day delivery for standard parts
- Same-day for emergency (with approval)
- Track all parts usage per job

## Customer Interaction

### Professional Standards
- Arrive within appointment window
- Introduce yourself and show ID
- Explain work to be performed
- Keep work area clean
- Explain what was done
- Ask if customer has questions

### Difficult Situations
- Remain calm and professional
- Listen to customer concerns
- Escalate to supervisor if needed
- Document all interactions
- Never argue with customer

### Customer Satisfaction
- Target NPS: 9+ for all visits
- Request feedback at job completion
- Address concerns before leaving
- Follow up if customer unhappy
